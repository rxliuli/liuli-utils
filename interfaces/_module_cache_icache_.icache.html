Interface ICache<T>

缓存接口 功能

  1. add 增加。如果不存在则添加，否则忽略
  2. del 删除。如果存在则删除，否则忽略
  3. set 修改。如果存在则设置，否则忽略
  4. get 根据 key 获取。如果存在则获取，否则忽略
  5. touch 根据 key 获取并刷新超时时间
  6. find 根据谓词查询 key
  7. list 根据谓词查询 key 获得列表
